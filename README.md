# Projecta.Backend
Backend de sistema de gerenciamento de projetos, laboratórios e grupos de pesquisas para universidades

Utilizando NodeJS, com Express como framework para desenvolvimento backend, este sistema permitirá aos usuários um ambiente centralizado para coordenar

## Tecnologias

- Express: Framework minimalista para desenvolvimento web backend;
- Sequelize: ORM para bancos de dados relacionais;
- PostgreSQL: Banco de dados relacionais baseado em SQL;
